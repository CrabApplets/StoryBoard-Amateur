# Storyboard Amateur Program

A beginner-friendly storyboard creation tool inspired by **StoryBoard Pro** by Atomic Learning. This program provides a simple and intuitive interface for creating storyboards and planning visual storytelling sequences.

---

## Features

- **Basic Storyboard Tools**: Create simple storyboards for videos.
- **Timeline**: You can re-order the timeline elements as desired.
- **Export Options**: Save your storyboard as a .json for easy storage or sharing or PDF for easy printing.
- **Themes**: There are various included themes that show in the timeline as well as in the PDF.
---

### Prerequisites

Before running the program, make sure you have the following libraries installed:

- **Tkinter**: Used for building the graphical user interface.
- **Pillow**: For handling and manipulating images.
- **JSON**: To handle reading and writing JSON files.
- **CSV**: For working with CSV files (for data storage/export).
- **datetime**: To manage timestamps and date-related operations.
- **os**: For file and directory manipulation.
- **typing**: Provides support for type hinting, such as `List`, `Dict`, and `Optional`.
